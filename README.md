# CineLog

CineLog e uma aplicacao web simples para organizar filmes que o usuario deseja assistir ou ja assistiu. O projeto foi feito com Vite e JavaScript puro, pensando em um aluno iniciante/intermediario.

## Funcionalidades
- Adicionar filmes com titulo, genero, nota e status assistido
- Editar filmes cadastrados
- Remover filmes
- Buscar filmes pelo titulo
- Calcular a media das notas
- Salvar e carregar dados com LocalStorage

## Tecnologias usadas
- HTML
- CSS
- JavaScript (ESM)
- Vite

## Como instalar
1. Tenha o Node.js instalado (ele inclui o npm).
2. Abra o terminal na pasta do projeto.
3. Rode o comando:

```bash
npm install
```

## Como executar
Para iniciar o servidor de desenvolvimento:

```bash
npm run dev
```

Depois, abra o link que o Vite mostrar no terminal.

## Como usar
1. Preencha o formulario com os dados do filme.
2. Clique em "Salvar" para adicionar.
3. Use os botoes de editar e remover nos cards.
4. Marque "Assistido" para atualizar o status.
5. Use a busca para filtrar por titulo.

## Requisitos da atividade (resumo)
- CRUD completo de filmes
- Programacao funcional com map, filter e reduce
- LocalStorage para persistencia
- Manipulacao do DOM sem frameworks
- Eventos com addEventListener
- ESM com import/export
- Interface simples, responsiva e com tema escuro
